# LearnSphere

A full-stack online learning platform enabling users to enroll in courses, watch video lectures, take quizzes, track progress, and receive certificates upon completion. Built with modern technologies like React, Node.js, and Supabase to deliver a scalable and interactive educational experience.

## Features

- User authentication (JWT / Supabase Auth)
- Course creation and management
- Video lessons streaming with secure access
- Quiz module with scoring and evaluation
- Real-time progress tracking
- Payment integration with Stripe/Razorpay (optional)
- Admin dashboard for monitoring users, courses, and revenue

## Tech Stack

| Layer         | Technology              | Purpose                                     |
| ------------- | ----------------------- | ------------------------------------------- |
| Frontend      | React.js, Tailwind CSS, React Router | Building responsive UI and routes           |
| Backend       | Node.js, Express.js     | RESTful APIs and business logic            |
| Database      | Supabase (PostgreSQL) or MongoDB | Data storage and authentication             |
| Authentication| JWT or Supabase Auth    | Secure login and user management            |
| Payment       | Stripe / Razorpay       | Handle course purchases                      |
| Deployment    | Vercel (Frontend), Render/Railway (Backend) | Hosting and continuous deployment |
